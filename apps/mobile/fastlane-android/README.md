# Android Fastlane

Fastlane config for Android release deploys. Lives outside `apps/mobile/android/` because that directory is gitignored (regenerated by `expo prebuild`).

The GitHub Actions workflow `.github/workflows/android-deploy.yml` runs `expo prebuild --platform android`, copies this folder into `apps/mobile/android/fastlane/`, copies the `Gemfile` into `apps/mobile/android/`, then runs `bundle exec fastlane deploy_internal`.

## Required secrets

| Secret | Purpose |
|---|---|
| `EXPO_PUBLIC_API_URL` | API URL baked into the build |
| `ANDROID_KEYSTORE_BASE64` | Upload keystore, base64-encoded |
| `ANDROID_KEYSTORE_PASSWORD` | Keystore password |
| `ANDROID_KEY_ALIAS` | Key alias inside the keystore |
| `ANDROID_KEY_PASSWORD` | Key password |
| `PLAY_STORE_SERVICE_ACCOUNT_JSON` | Google Play API service account JSON |

## Manual trigger

`gh workflow run android-deploy.yml`
